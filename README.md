## Development

You will need

- Latest Node.js LTS (https://nodejs.org/en)
- pnpm (https://pnpm.io)
- Rust (https://rustup.rs)
- Tauri CLI (after installing Rust) `cargo install tauri-cli`

Once you've installed Node.js, pnpm, and Rust, run `pnpm i` at the root of the repository.
